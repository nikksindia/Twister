## Requirements

- iOS 9.0+
- Xcode 8.1

## Installation

#### CocoaPods
You can use [CocoaPods](https://cocoapods.org/) to install `Twister` by adding it to your `Podfile`:

```ruby
platform :ios, '9.0'
use_frameworks!
pod 'Twister'
```

#### Manually
1. Add ```Alamofire``` as dependency in your project.
2. Download and drop ```Twister.swift``` in your project.  
3. Congratulations!
